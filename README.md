# JavaSecInterview

## 介绍

这是什么：Java安全研究与安全开发面试题总结

为什么要做：**帮助自己校招找到工作，同时帮助广大Java安全师傅顺利找到工作**

项目目标：完全掌握本项目后进轻松大厂

计划定期更新，从基础到各种实战问题，打造齐全的Java安全面试题库（包含问题和详细的答案）

作者技术水平有限，难免有错误之处，欢迎师傅们提出ISSUE和PR
